# m.cswendu.com
##基于Vue CLI3.0开发

#### 开发规范

- 1.前后端分离
- 2.公共接口请求地址
- 3.公共图片引用地址
- 4.进入页面验证token
- 5.console.log封装，仅在调试模式下打印
- 5.基于vuex进行状态管理
